# PDM-Hub ☑️📊
**A concept portal for Partner Development Managers (PDMs) built w/Claude by Anthropic.**

Self-contained, editable, or connect it to Dynamics CRM, SharePoint and other SaaS services through the REST API.

Align the objectives of your organization with your partners' through four interactive modules:

1) **Pipeline Dashboard** — Co-sell deal tracker across all five SME&C solution areas (App & Infra, Data & AI, Modern Workplace, Business Apps, Security) with RAG status, ARR values, consumption vs target progress bars, and a live deal entry form. Mirrors the JD's language around "monitoring top deals and consumption targets across all solution areas."

2) **QBR Command Centre** — Gap analysis table by solution area, open blocker/escalation logger with priority tagging, agenda builder, and a one-click Export QBR Summary that generates a clean exec-ready HTML report — directly demonstrating the "drive MBRs/QBRs, tracking execution, identifying gaps" responsibility.

3) **Copilot & AI Readiness Benchmark** — Four-dimension assessment (Sales Capability, Technical Certifications, AI Pipeline, Consumption) with live sliders, an auto-updating readiness tier (Foundational / Developing / Advanced), and a prioritized action plan that dynamically ranks dimensions by weakness and serves relevant next steps.

4) **GTM Co-Sell Planner** — Four-stage campaign builder (Readiness → Activation → Field Landing → Review) with interactive checklists, Microsoft incentive/program checklist (MCI, co-op funds, FastTrack, solution designations), and a milestone calendar.
