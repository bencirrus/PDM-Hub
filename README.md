# PDM-Hub

PDM Partner Intelligence Hub is a PWA high-value concept with four interactive modules:

Pipeline Dashboard — Co-sell deal tracker across all five SME&C solution areas (App & Infra, Data & AI, Modern Workplace, Business Apps, Security) with RAG status, ARR values, consumption vs target progress bars, and a live deal entry form. Mirrors the JD's language around "monitoring top deals and consumption targets across all solution areas."

QBR Command Centre — Gap analysis table by solution area, open blocker/escalation logger with priority tagging, agenda builder, and a one-click Export QBR Summary that generates a clean exec-ready HTML report — directly demonstrating the "drive MBRs/QBRs, tracking execution, identifying gaps" responsibility.
Copilot & AI Readiness Benchmark — Four-dimension assessment (Sales Capability, Technical Certifications, AI Pipeline, Consumption) with live sliders, an auto-updating readiness tier (Foundational / Developing / Advanced), and a prioritised action plan that dynamically ranks dimensions by weakness and serves relevant next steps.

GTM Co-Sell Planner — Four-stage campaign builder (Readiness → Activation → Field Landing → Review) with interactive checklists, Microsoft incentive/programme checklist (MCI, co-op funds, FastTrack, solution designations), and a milestone calendar.

The signal this sends to the hiring manager: You understood the role deeply enough to build the tools a PDM needs to actually do it — before your first day.
